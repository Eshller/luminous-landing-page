# Adzzat Landing Page

A stunning, modern landing page featuring Soul AI-inspired design, premium 3D animations, and enterprise-grade UI built with React, TypeScript, and Framer Motion.

## Features

- **Premium Visual Experience**: 3D animations, floating orbs, glassmorphism effects
- **Enterprise AI Services**: Hire Talent, LLM Evaluation, Dataset Creation, Red Teaming
- **Responsive Design**: Optimized for all devices
- **Modern Tech Stack**: React 18, TypeScript, Framer Motion, Tailwind CSS

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd luminous-landing-page

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Tech Stack

This project is built with:

- Vite
- TypeScript
- React 18
- Framer Motion
- shadcn-ui
- Tailwind CSS

## Build for Production

```sh
npm run build
```
